1. Static The default position value for elements is static, where they are positioned according to the normal flow of the document.
This is a static position element.
2. Fixed Elements with position: fixed are positioned relative to the viewport and remain fixed even when the page is scrolled.
This is a fixed position element.
3. Sticky Elements with position: sticky are positioned based on the normal flow of the document until a specified scroll position is reached. Then, they become "sticky" and remain fixed in that position.
This is a sticky position element.
4. Relative Elements with position: relative are positioned relative to their normal position in the document flow. Adjusting the top, bottom, left, or right properties can move them from their original position.
This is a relative position element.
5. Absolute Elements with position: absolute are positioned relative to the nearest positioned ancestor (an element with a position value other than static) or the containing block. They are taken out of the normal flow and can overlap other elements.
This is an absolute position element.
